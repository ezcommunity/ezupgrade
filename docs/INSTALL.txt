INSTALLATION

* Check out ezupgrade to the web root of where the upgraded installation will reside.

* Check out ezcopy to the /lib folder of ezupgrade (making the directory
  /lib/ezcopy) and and follow its installation instructions. Finally, copy the
  ezcopy.ini file from lib/ezcopy/settings/ to settings/  
  
* Copy settings/account_default.ini and rename the copy settings/account.ini
  For each account you need to upgrade, create an Account block with the 
  proper settings.


CONFIGURATION

* chmod 777 ezupgrade


USAGE

Run

php ezupgrade [account_username]


ASSUMPTIONS

DB dumps are assumed to be in a directory under the eZ root called "_dbdump/" and 
have the name "[db_name].sql".
